# joystick_virtual
## Requerimientos
Para que funcione, la red (internet) debe ser la misma en los equipos, el $ROS_DOMAIN_ID debe ser el mismo, y no deben de tener activado algún firewall.
## Ejecución
Ejecutar con
```bash
python3 virtual_joy_node.py
```
o bien con
```bash
python virtual_joy_node.py
```
